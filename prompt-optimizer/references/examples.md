# Prompt Optimizer Examples

## Example 1 — Recruitment Copy | 招聘文案

### Original | 原始输入
帮我写个招聘文案

### Optimized Direction | 优化方向
- 明确岗位
- 明确公司阶段
- 明确目标候选人
- 明确输出结构
- 明确语气和长度

## Example 2 — Excel Cleaning Script | Excel 清洗脚本

### Original | 原始输入
写个 Python 脚本清理 Excel

### Optimized Direction | 优化方向
- 明确 Python + pandas
- 明确常见清洗动作
- 明确输入输出
- 要求注释和运行方式

## Example 3 — English Launch Email

### Original
write a launch email

### Optimized Direction
- define product type
- define audience
- define email goal
- define structure
- define tone and CTA
